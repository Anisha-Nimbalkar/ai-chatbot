# AI Chatbot using Flask & OpenAI API

## Overview
This project is a simple AI chatbot built using Python, Flask, and OpenAI API. It processes user queries and generates intelligent responses.

## Features
- REST API for chatbot interaction
- AI-based response generation
- Simple and scalable backend

## Tech Stack
- Python
- Flask
- OpenAI API

## How to Run
1. Install dependencies:
   pip install flask openai
2. Run:
   python app.py
